# In Silico Study On Dermal Interaction Of Bioactive Compounds

## Overview

This project focuses on studying the dermal permeability and diffusion behavior of rice-derived bioactive compounds using computational approaches, machine learning models, and VCell-based simulations.

The study evaluates how molecular descriptors such as molecular weight, LogP, TPSA, and hydrogen bonding properties influence skin permeability and transdermal diffusion.

---

## Objectives

* Analyze dermal permeability behavior of rice-derived bioactive compounds
* Predict skin permeability (logKp) using machine learning approaches
* Simulate diffusion across multilayer skin structures using VCell
* Identify compounds with improved topical and cosmetic application potential

---

## Methodology

1. Collection of rice-derived bioactive compounds from literature and databases
2. Molecular descriptor extraction and preprocessing
3. Machine learning-based permeability prediction
4. Development of multilayer skin diffusion model using VCell
5. Analysis of diffusion profiles and permeability trends

---

## Tools & Technologies

* Python

## Machine Learning Models Used

* Linear Regression
* Random Forest Regression
* Support Vector Regression (SVR)
* Gradient Boosting

---

## Key Findings

* Molecular weight, LogP, and TPSA strongly influence skin permeability
* Ensemble learning models showed improved prediction performance
* VCell simulations successfully modeled diffusion across skin layers
* Certain rice-derived bioactives demonstrated favorable dermal permeation behavior
